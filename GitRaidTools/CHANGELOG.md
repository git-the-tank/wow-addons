## 1.0.0 — Initial Release

- Raid countdown timer with configurable schedule and Edit Mode support
- Guild invite announcer with rotating flavor text
- Dispatch status frame showing automated raid-day action checklist
- Gear audit: scan raid for enchants, gems, tier pieces, and item levels
- Three audit views: Enchants, Tier Sets, Item Level (with difficulty track icons)
- Audit report: whisper summary of issues to selected players
- AceConfig settings panel with 5 tabs (General, Invites, Countdown, Dispatch, Audit)
- Optional MRT integration for mass guild invites
- LibEditMode integration for countdown frame positioning
