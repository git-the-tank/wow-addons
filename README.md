# WoW Addons

World of Warcraft addons for retail (Midnight / 12.x).

## Addons

- **GitRaidTools** — Raid utility toolkit: pull countdown timer, raid invite automation
- **TankBattleText** — Tank combat text and DTPS tracking with Edit Mode support
- **SimpleRaidTargetIcons** — Quick raid target icon assignment widget
- **RCLootCouncil_CouncilRotation** — RCLootCouncil plugin for rotating loot council members

## Tools

- `gen-bigwigs-colors.py` — Generate BigWigs bar color overrides from a classification mapping
- `fetch-spell-tooltips.py` — Fetch and cache spell tooltips from Wowhead for color audit
- `lint.sh` — Run luacheck static analysis on addon(s)
- `sync.sh` / `watch.sh` — Sync addon(s) to WoW AddOns folder (WSL → Windows)
